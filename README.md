# Uzbekistan Service - Мобильное приложение

Мобильное приложение для соединения клиентов со специалистами в Узбекистане.

## Описание

Uzbekistan Service - это платформа, которая соединяет клиентов с местными специалистами для оказания услуг на дому. Приложение поддерживает следующие категории услуг:

- **Барберы** - стрижки, бритье, уход за бородой
- **Няни** - уход за детьми, присмотр
- **Мастера по дому** - электрик, сантехник, мелкий ремонт
- **Строительные бригады** - строительные и ремонтные работы

## Функционал

### Для клиентов:
- Регистрация и вход через SMS
- Поиск специалистов по категориям
- Просмотр профилей специалистов
- Создание заказов с указанием адреса и времени
- Оплата наличными
- Оценка и отзывы о специалистах

### Для специалистов:
- Регистрация и создание профиля
- Загрузка фото и описания услуг
- Установка цен
- Получение уведомлений о заказах
- Управление заказами
- Просмотр статистики и заработков

## Технологии

- **Flutter** - кроссплатформенная разработка
- **Riverpod** - управление состоянием
- **Go Router** - навигация
- **Hive** - локальное хранение данных
- **Dio** - HTTP клиент
- **Firebase** - аутентификация и push-уведомления

## Установка и запуск

### Требования
- Flutter SDK 3.0.0 или выше
- Dart SDK 3.0.0 или выше
- Android Studio / Xcode для эмуляторов

### Установка зависимостей
```bash
flutter pub get
```

### Генерация кода
```bash
flutter packages pub run build_runner build
```

### Запуск приложения
```bash
# Для разработки
flutter run

# Для релиза
flutter build apk --release
flutter build ios --release
```

## Структура проекта

```
lib/
├── constants/          # Константы приложения
├── models/            # Модели данных
├── providers/         # Провайдеры состояния
├── screens/           # Экраны приложения
├── services/          # Сервисы для работы с API

└── widgets/           # Переиспользуемые виджеты
```

## API

Приложение работает с REST API, расположенным по адресу:
- Development: `http://localhost:3000/api/v1`
- Production: `https://api.uzbekservice.uz/api/v1`

## Многоязычность

Приложение поддерживает три языка:
- Русский (по умолчанию)
- Узбекский
- Английский
