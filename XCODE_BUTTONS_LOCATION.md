# 📍 Где найти кнопки Product и Archive в Xcode

## 🎯 Меню Product

### Расположение:
**В самой верхней строке Xcode** (рядом с логотипом Apple 🍎)

```
┌─────────────────────────────────────────────────────────┐
│ 🍎  File  Edit  View  Navigate  Editor  Product  Debug │ ← ВОТ ЗДЕСЬ!
└─────────────────────────────────────────────────────────┘
```

### Как найти:
1. Посмотрите **в самый верх экрана Xcode**
2. Найдите слово **"Product"** (между "Editor" и "Debug")
3. **Нажмите** на "Product"

## 📦 Кнопка Archive

### Расположение:
**Внутри меню Product** (после нажатия на "Product")

```
Product ▼
├─ Build (⌘B)
├─ Clean Build Folder...
├─ Run (⌘R)
├─ Test (⌘U)
├─ Profile (⌘I)
├─ Analyze (⌘⇧B)
├─ Archive ← ВОТ ОНА!
└─ ...
```

### Как найти:
1. Нажмите на **"Product"** в верхнем меню
2. В выпадающем меню найдите **"Archive"**
3. **Нажмите** на "Archive"

## 📋 Полная последовательность (пошагово):

### Шаг 1: Выберите устройство
```
В верхней панели Xcode (где кнопка Play ▶️):
[▶️ Play] [Any iOS Device ▼] [⚙️]
              ↑
        Выберите "Any iOS Device"
```

### Шаг 2: Откройте меню Product
```
Верхняя строка меню:
File | Edit | View | ... | Editor | [Product] | Debug
                                    ↑
                              Нажмите здесь
```

### Шаг 3: Выберите Archive
```
После нажатия на Product откроется меню:
┌─────────────────┐
│ Build           │
│ Clean...        │
│ Run             │
│ Test            │
│ Profile         │
│ Analyze         │
│ Archive  ← НАЖМИТЕ СЮДА
│ ...             │
└─────────────────┘
```

### Шаг 4: Дождитесь завершения
- Xcode начнет сборку
- Внизу появится прогресс-бар
- Займет 5-10 минут
- После завершения откроется **Organizer**

## 🖼️ Визуальная схема всего экрана:

```
┌─────────────────────────────────────────────────────────┐
│ 🍎 File Edit View Navigate Editor [Product] Debug ...  │ ← Шаг 2: Нажмите Product
├─────────────────────────────────────────────────────────┤
│ [▶️] [Any iOS Device ▼] [⚙️] [🔍]                     │ ← Шаг 1: Выберите устройство
├─────────────────────────────────────────────────────────┤
│                                                          │
│                    [Код проекта]                        │
│                                                          │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## ⚠️ Важно:

1. **Перед Archive** обязательно выберите **"Any iOS Device"** (не симулятор!)
2. Если кнопка Archive **неактивна** (серая), проверьте:
   - Выбрано ли "Any iOS Device"
   - Настроено ли Signing & Capabilities
   - Есть ли ошибки в проекте

## 🔍 Альтернативный способ:

Если не можете найти меню Product:
1. Попробуйте **горячие клавиши**: `Cmd + Shift + B` (может открыть Archive напрямую)
2. Или сначала: **Product → Build** (`Cmd + B`), затем **Product → Archive**

## 📱 После Archive:

1. Откроется окно **Organizer**
2. Выберите ваш архив
3. Нажмите **"Distribute App"**
4. Следуйте инструкциям для загрузки в TestFlight
