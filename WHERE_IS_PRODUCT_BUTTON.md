# 📍 ГДЕ НАЙТИ КНОПКУ PRODUCT В XCODE

## 🎯 РАСПОЛОЖЕНИЕ:

### В САМОЙ ВЕРХНЕЙ СТРОКЕ ЭКРАНА!

```
┌─────────────────────────────────────────────────────────┐
│ 🍎 Xcode  File  Edit  View  Find  Navigate  Editor     │
│     [Product]  Debug  Integrate  Window  Help          │
│          ↑                                              │
│      ВОТ ЗДЕСЬ!                                        │
└─────────────────────────────────────────────────────────┘
```

## 📋 ПОШАГОВАЯ ИНСТРУКЦИЯ:

### Шаг 1: Посмотрите ВВЕРХ
- Поднимите глаза в **самую верхнюю часть экрана**
- Это строка меню macOS (рядом с логотипом Apple 🍎)

### Шаг 2: Найдите слово "Product"
- В строке меню найдите: **File | Edit | View | ... | Editor | Product | Debug**
- **Product** находится между **"Editor"** и **"Debug"**

### Шаг 3: Нажмите на "Product"
- **Кликните** на слово **"Product"**
- Откроется выпадающее меню

### Шаг 4: Выберите "Archive"
- В меню найдите **"Archive"**
- **Кликните** на **"Archive"**

После нажатия на "Product" откроется:

```
┌─────────────┐
│ Build       │
│ Clean...    │
│ Run         │

│ ...         │
└─────────────┘
```

## ⚠️ ВАЖНО:

1. **Product** - это НЕ кнопка в панели инструментов
2. **Product** - это пункт меню в ВЕРХНЕЙ СТРОКЕ
3. Нужно кликнуть на слово "Product" в меню

## 🔍 ЕСЛИ НЕ ВИДИТЕ:

1. **Убедитесь, что окно Xcode развернуто**
2. **Посмотрите в самую верхнюю строку** (рядом с 🍎)
3. **Product** находится между "Editor" и "Debug"
4. Если не видно, попробуйте развернуть окно на весь экран

## ⌨️ АЛЬТЕРНАТИВА (горячие клавиши):

- **Product → Build**: `Cmd + B`
- **Product → Clean Build Folder**: `Shift + Cmd + K`
- **Product → Archive**: нет прямой горячей клавиши, нужно через меню

## 📝 ПОЛНАЯ ПОСЛЕДОВАТЕЛЬНОСТЬ:

1. ✅ Посмотрите ВВЕРХ (в строку меню)
2. ✅ Найдите "Product" (между Editor и Debug)
3. ✅ Нажмите на "Product"
4. ✅ В меню выберите "Archive"
5. ✅ Дождитесь завершения (5-10 минут)
